1. アプリの概要

「PyLingo」は​​、プログラミング学習中の「わからない」をその場で解決するデスクトップ特化型のAIアシスタントです。

日本語の難解なエラーメッセージや専門用語を、プログラミング初心者向けにわかりやすい日本語に翻訳・解説します。

外部のインターネット通信を一切行わず、お使いのPC内（完全ローカル環境）で最新の統合AIが動作するため、APIキーの登録不要で、情報漏洩のリスクなく安全にお使いいただけます。

2.動作環境

OS: Windows 10 / Windows 11

通信環境：不要（完全オフラインで動作します）

メモリ(RAM)：8GB以上推奨（AIモデルを読み込むため、起動メモリを使用します）

3. 準備と起動手順

本アプリはインストール不要ですぐにお使いいただけます。

ダウンロードしたフォルダの中にあるapp.exeをダブルクリックして起動してください。

【重要】app.exeと同じ場所にあるモデルフォルダは、絶対に移動したり削除したりしないで。AIの脳みそとなるデータが入っています。

初回起動時、AIモデルをPCのメモリに読み込むため、画面が上がるまでに数〜十数秒ほど時間がかかります。（PCのスペックにより変動します）そのまま少しお待ちください。

4.基本的な使い方

画面上のスイッチで、用途に合わせて2つのモードを開くことができます。

🔤単語解説モード

コードの中で見つけた見慣れない英単語（変数、関数名など）を入力してください。

⚠️エラー解決モード

ターミナルに出た赤いエラーメッセージ（トレースバックなど）をそのまま複数行コピーして貼り付けてください。エラーの直訳、原因の箇条書き、および具体的な解決策（修正コード）を提案します。

📌 画面の最後のピン留め機能

右上の「📌固定」ボタンを押すと、PyLingoの画面が常に一番手前に表示されます。VS Codeなどのエディタでコードを書きながら、邪魔にならない位置に辞書の代わりに活用してください。
